# RAP的使用流程

## 登录

目前只支持 `域帐号` 登录。

域帐号登录后会自动完成新用户注册及登录流程。

## 创建项目

- 选择 `产品线` （产品线由系统管理员内置，如果需要添加，请联系系统管理员）

- 选择 或 创建 `分组`

- 在分组下创建项目

## 接口编辑

- 新建接口

- 增加接口的请求参数

- 增加接口的相应参数

- 保存

## 接口使用

- 在Kissy或jQuery下，引入rap-plugin.js 插件

- 配置RAP
